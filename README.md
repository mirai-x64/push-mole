叩いてもモグラは死なない。隣の穴へ飛ぶだけ。
**モグラ同士をぶつけた時だけ消える。**

2003年頃のケータイの白黒液晶プリインゲーム風。
モグラは「未読の着信」で、消えると「既読」になる。

## 遊び方

- 穴を叩くと、中のモグラが**叩いた点の反対側**へ隣の穴 1 マスぶん飛ぶ
  (左を叩けば右へ飛ぶ)。カーソルを乗せると飛ぶ向きと行き先が出る。
- 飛んだ先が空なら居座る。**先にモグラがいれば 2匹とも弾けて消える。**
- **盤外へは飛ばない。** 壁にぶつかって元の穴に落ちる(向きが変わるだけの空振り)。
- 空の穴を叩いても何も起きない。
- 1.6 秒ごとに着信が増える。**全16穴が埋まったら終了。**

## 設計上の一点

モグラを盤から除去する手段は**衝突ただ一つ**に固定してある。

端のモグラを盤外へ弾き出して減らせると、そちらが常に 1 手で済むため、
「空の穴へ寄せる → 隣から押し込む」という 2 手の組み立てが誰も選ばない手になる。
盤外を壁にして逃げ道を塞ぐことで、盤を減らす唯一の道が
**2匹を隣り合わせにすること**になり、そこで初めて考える意味が生まれる。

`test/play.mjs` の「単独のモグラは64通りどう押しても消えない」が、
この一点を全マス×全方向の総当たりで押さえている。

## 検証

実際の Chrome を CDP で動かし、本物の click を投げて遊ばせている。依存なし。

```
node test/play.mjs    # 27項目の検証
node test/shot.mjs    # /tmp/mole-*.png に画面を書き出す
```

成功条件に対応する検証:

| 条件 | 検証 |
| --- | --- |
| 叩いた向きへ1マス飛ぶ | 4方向 + 飛び越さないこと |
| 盤外へ出ない | 盤の縁6通り + 全64通りの総当たり |
| 狙って2匹を消せる | 横/縦/斜めを2手、隣接を1手、3匹目を巻き込まないこと |
| 放置すると埋まって終了 | 実時間で16穴埋まるまで待つ |
